express で socket.io をつかって見たサンプルです。

npm install
npm start

複数のブラウザで http://localhost:3000 を開き、
マウス移動や、キーボード雄下をしてみてください。
フォーカスが当たっていないブラウザの画面中でも接続車数、キーのコード、マウス位置の表示が更新されます。

https://github.com/sksmatt/nodejs-cursor-tracking を参考にして作成しました。
